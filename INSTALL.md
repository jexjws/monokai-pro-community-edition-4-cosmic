# Monokai Pro (Community Edition) for [COSMIC desktop](https://system76.com/cosmic)

## Installation instructions

Download the latest `.ron` theme files from the [Releases page](https://github.com/jexjws/monokai-pro-community-edition-4-cosmic/releases):

### Activating COSMIC Desktop theme

1. COSMIC Settings → Desktop → Appearance → Import，select `monokai-pro-ce-cosmic.ron`
2. Enjoy!

### Activating COSMIC Terminal theme

1. COSMIC Terminal → View → Colour schemes → Import，select `monokai-pro-ce-cosmic-terminal.ron`
2. Select "Monokai Pro (CE)" from the theme list in View → Settings
3. Enjoy!


> **Note:** COSMIC Terminal profiles can override the global terminal color scheme. If the colors do not change after selecting this scheme in View → Settings, open View → Profiles... and check whether a profile is marked Make default. Please set that profile’s Colour scheme dark/light to Monokai Pro (CE),or disable Make default. See also [pop-os/cosmic-term#855](https://github.com/pop-os/cosmic-term/issues/855).
